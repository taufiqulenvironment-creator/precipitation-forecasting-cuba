# Forecasting Precipitation in Cuba Using Graph Neural Networks

## Overview

This repository contains the source code, documentation, and results from my Master's research conducted at **The University of Texas at Arlington**.

The project develops **Graph Convolutional Recurrent Neural Networks (GCRNN)** and **Long Short-Term Memory (LSTM)** models to forecast daily precipitation across Cuba using NOAA CPC Unified precipitation observations.

---

## Objectives

- Develop graph-based deep learning models for precipitation forecasting.
- Compare GCRNN with conventional LSTM models.
- Evaluate model performance using unseen monitoring locations.
- Quantify predictive uncertainty using Monte Carlo Dropout.

---

## Dataset

- NOAA CPC Unified Daily Precipitation
- Study Area: Cuba
- Time Period: 1979–2023
